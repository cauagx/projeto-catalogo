# Projeto Catálogo de Itens - Squad [Cauã Victor, Carlos Guilherme]

Este projeto foi desenvolvido como requisito final da disciplina, simulando o fluxo de trabalho de uma squad de alto desempenho. O foco é a implementação de praticas modernas de **devops**, utilizando conteinerização e integração contínua.

## Tecnologias Utilizadas 
-**Backend:** Python (FastAPI)
-**Frontend:** HTML/JS puro
-**Containerização:** Docker
-**Automação** GitHub Actions (CI/CD)
